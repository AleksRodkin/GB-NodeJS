# Функция решения квадратного уравнения

Функция принимает действительные числа `'a', 'b' и 'c'` из квадратного уравнения вида `ax^2 + bx + c = 0`

Первым шагом вычисляется дискриминант по формуле D = b * b - 4 * a * c;

Если дискриминант меньше нуля, значит корней у данного уравнения нет, фунция ничего не возвращает.
Если дискриминант равен нулю, корень у уравнения один, функция возвращает одно значение.
Если дискриминант - положительное число, уравнение имеет два корня, функция возвращает массив с двумя значениями.

Для наглядности в каждом из условий указан вывод в консоль соответствующего решения.
